# Limitations

The included evidence, reset timeline, analogs, and backtests are synthetic demonstration fixtures. Expert priors are judgmental and untrained. A single public account is a sparse source. X API access, Supabase persistence, embeddings, signed Resend webhook persistence, retry queues, and real deliverability are not exercised without credentials. Forecasts are experimental, not promises about product behavior.

The cinematic portraits are intentionally absent. Supply licensed files at `public/cinematic/tibo-opening.webp` and `public/cinematic/tibo-ending.webp`; until then the hero uses the existing neutral placeholder.
